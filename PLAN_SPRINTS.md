# Sprint Plan for DepiMovil

## Sprint 0: Setup
- **Tasks:**
  1. Establish development environment.
  2. Set up project repository.
  3. Define team roles and responsibilities.
- **Acceptance Criteria:**
  - Development environment operational.
  - Repository accessible to all team members.
  - Team roles documented and agreed upon.
- **Expected Output:**
  - Fully configured development environment.
  - Initialized project repository on GitHub.
  - Document outlining team roles.
- **Timeline:**
  - Start Date: 2026-03-16 
  - End Date: 2026-03-22

## Sprint 1: Auth + Operadoras
- **Tasks:**
  1. Implement user authentication module.
  2. Integrate operator data for services.
  3. Create API endpoints for authentication.
- **Acceptance Criteria:**
  - User can register, login, and logout.
  - Operators' data retrieved and displayed.
  - Authentication API responses are secure and correct.
- **Expected Output:**
  - Functional authentication system.
  - API documentation for authentication.
- **Timeline:**
  - Start Date: 2026-03-23 
  - End Date: 2026-03-29
