# Wireframes for DepiMovil

## Login Page
- **Layout**: Full-screen with a centered form.
- **Components**:
  - Logo at the top.
  - Input fields for username and password.
  - Login button.
  - Link to "Forgot Password".
- **User Flow**: Users enter credentials and click login to access the Dashboard.
- **Field Specifications**:
  - Username: Required, text.
  - Password: Required, password type.

## Dashboard (Admin)
- **Layout**: Grid layout with navigation sidebar.
- **Components**:
  - Sidebar with menu items (Operadoras, Máquinas, Reservas, Pagos, Incidencias).
  - Main area displaying statistics.
- **User Flow**: Admin can navigate to different sections using the sidebar.

## Operadoras List
- **Layout**: Table layout.
- **Components**:
  - Search bar at the top.
  - Table columns (Name, Contact, Actions).
  - Add new Operadora button.
- **User Flow**: Admin can search and add Operadoras.

## Máquinas List
- **Layout**: Similar to Operadoras List, in tabular format.
- **Components**:
  - Search bar.
  - Table (ID, Name, Location, Status).
  - Add new Máquina button.
- **User Flow**: Display and manage machines available in the system.

## Reservas List
- **Layout**: Table with filters.
- **Components**:
  - Filters for date, user, status.
  - Table (ID, User, Date, Status, Actions).
- **User Flow**: Admin can filter and view reservations.

## Pagos Form
- **Layout**: Form layout.
- **Components**:
  - Input fields for amount, user, date, payment method.
  - Submit button.
- **User Flow**: Admin inputs payment details and submits to log a payment.

## Incidencias
- **Layout**: Stack layout for incidents reported.
- **Components**:
  - List of incidents (ID, Description, Status).
  - Button to add new Incidencia.
- **User Flow**: Admin views and manages incidents reported by users.

---

This document outlines the wireframes for the main user interfaces in DepiMovil, providing a comprehensive look at each screen's structure and functionalities.
