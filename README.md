## Introduction
DEPIMOVIL is a project aimed at providing an efficient solution for mobile device management. It allows users to manage their devices effectively through a user-friendly interface.

## Installation
To install DEPIMOVIL, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/profejorgeemi-art/DEPIMOVIL.git
   ```
2. Navigate into the project directory:
   ```bash
   cd DEPIMOVIL
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
You can then access the application at `http://localhost:3000`.
